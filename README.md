# BaiyunKeys 小程序项目简介

BaiyunKeys 是一个用于管理平安回家蓝牙门禁的微信小程序，面向需要在多终端同步门禁配置的用户。项目核心目标是提供安全、透明的本地化配置体验，帮助用户高效获取、保存并调试门禁参数。

## 核心能力

- 支持自动获取、数据库提取、抓包三种方式导入门禁参数；
- 通过蓝牙握手开锁，附带日志同步、快速开锁等调试功能；
- 提供门禁配置管理、草稿缓存、复制提醒等易用性增强。

## 使用指南
1. 首次打开小程序时，根据引导在「帮助」页阅读免责声明并选择合适的配置导入方式（自动获取、数据库提取或抓包）。
2. 若选择自动获取，请在「配置」页填写实名手机号与身份证号，完成接口调用后复制并备份生成的门禁参数。
3. 若选择数据库提取或抓包方式，请确保提取到 `MAC_NUM`、`PRODUCT_KEY`、`bluetoothName` 等关键字段，并在「配置」页完整录入。
4. 配置保存后可在首页选择对应门禁进行蓝牙开锁；如需更快体验，可在「配置」页开启快速开锁与日志同步。
5. 遇到连接异常时，可在首页查看实时日志，或通过「配置」页的复制提示导出门禁信息，方便排查与备份。
